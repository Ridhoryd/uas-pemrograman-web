Pertemuan 6 - Bullet Numbering dan List

Materi:

1. Unordered List (List Tidak Urut):

List ini digunakan untuk menampilkan data tanpa urutan tertentu. Biasanya, item dalam daftar diberi tanda bullet (â€¢). Berikut adalah contoh list tidak urut:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

2. Ordered List (List Terurut):

List terurut digunakan ketika urutan item penting, misalnya langkah-langkah instruksi. Berikut adalah contoh list terurut:

<ol>
    <li>Langkah 1</li>
    <li>Langkah 2</li>
    <li>Langkah 3</li>
</ol>

3. List dengan Kombinasi:

Anda juga dapat membuat kombinasi dari unordered dan ordered list. Berikut adalah contoh menggunakan nested list:

<ul>
    <li>Item 1</li>
    <li>Item 2
        <ol>
            <li>Langkah A</li>
            <li>Langkah B</li>
        </ol>
    </li>
</ul>
