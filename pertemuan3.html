Pertemuan 3 - Tabel dan Formulir HTML

Materi:

Di pertemuan ini, kita akan membahas cara membuat tabel dan formulir menggunakan HTML. Keduanya adalah elemen dasar yang sangat penting dalam pembuatan halaman web dinamis.

1. Tabel HTML:

Tabel digunakan untuk menyajikan data dalam bentuk baris dan kolom. Berikut adalah contoh elemen-elemen yang digunakan dalam pembuatan tabel:

  • <table>: Digunakan untuk mendefinisikan tabel.
  • <tr>: Digunakan untuk mendefinisikan baris tabel.
  • <th>: Digunakan untuk mendefinisikan header tabel.
  • <td>: Digunakan untuk mendefinisikan sel tabel.

Berikut adalah contoh kode tabel HTML:

<table>
    <tr>
        <th>Nama</th>
        <th>Usia</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Mary</td>
        <td>30</td>
    </tr>
</table>

2. Formulir HTML:

Formulir digunakan untuk mengumpulkan data dari pengguna. HTML menyediakan elemen-elemen untuk membuat berbagai jenis input data, seperti teks, radio buttons, checkbox, dan tombol kirim.

  • <form>: Digunakan untuk mendefinisikan formulir.
  • <input>: Digunakan untuk membuat elemen input seperti teks, radio buttons, checkbox, dan lainnya.
  • <textarea>: Digunakan untuk membuat area teks yang lebih besar.
  • <select>: Digunakan untuk membuat dropdown list.

Berikut adalah contoh kode formulir HTML:

<form action="submit_form" method="post">
    <label for="name">Nama</label>
    <input type="text" id="name" name="name" required>
    <label for="age">Usia</label>
    <input type="number" id="age" name="age">
    <button type="submit">Kirim</button>
</form>
