Pertemuan 1 - Pengenalan Website

Materi:

Di pertemuan pertama ini, kita akan membahas dasar-dasar mengenai **Website** dan **HTML**. Kita akan mengulas sejarah, cara kerja, dan komponen dasar yang ada dalam website.

1. Pengertian Website:

Website adalah sekumpulan halaman web yang dapat diakses melalui jaringan internet menggunakan browser. Website digunakan untuk berbagai tujuan, mulai dari berbagi informasi, berkomunikasi, transaksi bisnis, hingga hiburan. Contoh website termasuk Google, Facebook, dan Wikipedia.

2. Sejarah Web:

Internet pertama kali digunakan pada tahun 1960-an, namun pengembangan World Wide Web (WWW) dimulai pada tahun 1990 oleh Tim Berners-Lee. Tim menciptakan protokol pertama untuk komunikasi web, yaitu HTTP (Hypertext Transfer Protocol), serta HTML untuk membangun halaman web pertama.

Seiring berjalannya waktu, perkembangan web semakin pesat, dan kini kita dapat mengakses berbagai informasi dalam hitungan detik, menggunakan berbagai perangkat seperti laptop, smartphone, dan tablet.

3. Cara Kerja Website:

Website bekerja dengan memanfaatkan server dan client. Server adalah komputer yang menyimpan halaman web, sedangkan client adalah perangkat (seperti komputer atau smartphone) yang digunakan untuk mengakses halaman tersebut. Pengguna mengakses website melalui browser, yang kemudian mengirimkan permintaan ke server untuk menampilkan halaman yang diminta.

4. Komponen Utama dalam Website:

Website umumnya terdiri dari dua komponen utama:

  • Frontend: Bagian website yang dapat dilihat dan diakses oleh pengguna. Frontend mencakup elemen-elemen seperti teks, gambar, layout, dan interaktivitas yang ditangani menggunakan HTML, CSS, dan JavaScript.
  • Backend: Bagian yang tidak terlihat oleh pengguna, namun menangani logika dan pengolahan data. Backend melibatkan server, database, dan bahasa pemrograman server-side seperti PHP, Python, Ruby, dan lainnya.

5. Perbedaan Website Statis dan Dinamis:

Website dapat dibagi menjadi dua jenis berdasarkan cara kontennya ditampilkan:

  • Website Statis: Konten halaman tidak berubah kecuali ada pembaruan yang dilakukan secara manual. Website ini menggunakan HTML dan CSS untuk menampilkan halaman yang tetap.
  • Website Dinamis: Konten halaman dapat berubah secara otomatis berdasarkan data yang diberikan. Website dinamis menggunakan server-side scripting seperti PHP atau Node.js untuk menampilkan data yang diperbarui setiap waktu.

6. Fungsi dan Kegunaan Website:

Website memiliki berbagai fungsi, antara lain:

  • Berbagi Informasi: Website digunakan untuk berbagi berbagai informasi seperti berita, artikel, dan tutorial.
  • Media Sosial: Website sosial media seperti Facebook dan Twitter memungkinkan pengguna untuk berinteraksi dan berbagi konten.
  • Bisnis dan E-Commerce: Website dapat digunakan untuk berjualan produk atau jasa secara online melalui platform e-commerce seperti Tokopedia atau Shopee.
  • Pendidikan: Website pendidikan memungkinkan proses belajar mengajar berlangsung secara online, misalnya melalui platform seperti Google Classroom atau Coursera.

7. Tugas dan Latihan:

Setelah memahami materi ini, Anda diharapkan dapat:

  • Menjelaskan pengertian **Web**, **Portal**, dan **Blog**.
  • Mengidentifikasi istilah **Web Browser** dan memberi contoh.
  • Menjelaskan cara kerja browser dalam membaca script HTML.
