Pertemuan 2 - HTML Dasar

Materi:

Di pertemuan kedua ini, kita akan belajar mengenai dasar-dasar HTML yang digunakan untuk membangun halaman web.

1. Struktur HTML:

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Struktur dasar HTML terdiri dari beberapa elemen penting:

  • <html>: Elemen utama yang mengelilingi seluruh dokumen HTML.
  • <head>: Berisi informasi meta tentang dokumen, seperti judul halaman dan link ke file eksternal.
  • <body>: Bagian ini berisi konten yang ditampilkan di halaman web seperti teks, gambar, dan lainnya.

2. Tag Dasar HTML:

Beberapa tag dasar yang sering digunakan dalam HTML adalah:

  • <h1> hingga <h6>: Digunakan untuk membuat heading, dengan <h1> sebagai yang terbesar dan <h6> sebagai yang terkecil.
  • <p>: Digunakan untuk membuat paragraf teks.
  • <a>: Digunakan untuk membuat tautan atau hyperlink antar halaman.
  • <img>: Digunakan untuk menampilkan gambar di halaman web.
  • <ul>, <ol>, <li>: Digunakan untuk membuat daftar tanpa urutan (<ul>) atau dengan urutan (<ol>).

3. Penerapan HTML dalam Website:

HTML digunakan untuk membuat struktur halaman web, namun untuk mendesain tampilan dan interaktivitas, kita juga perlu menggunakan CSS dan JavaScript.
