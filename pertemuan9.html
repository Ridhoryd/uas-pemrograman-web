Pertemuan 9 - Tabel HTML

Materi:

1. Pengenalan Tabel HTML:

Tabel HTML digunakan untuk menampilkan data yang terstruktur dalam bentuk baris dan kolom. Tabel terdiri dari beberapa elemen utama, yaitu:

<table>
    <tr>
        <th>Nama</th>
        <th>Usia</th>
    </tr>
    <tr>
        <td>Ali</td>
        <td>25</td>
    </tr>
</table>

2. Atribut Tabel:

Atribut untuk tabel bisa digunakan untuk mengatur ukuran, garis batas, dan tampilan lainnya:

  • border: Menentukan apakah tabel akan memiliki batas.
  • cellspacing: Menentukan jarak antar sel dalam tabel.
  • cellpadding: Menentukan jarak antara konten dan batas sel.

3. Menggunakan Tabel untuk Tampilan Data:

Tabel sering digunakan untuk menampilkan data yang memiliki hubungan antar elemen. Misalnya, tabel yang berisi data karyawan, produk, atau informasi lainnya.

<table border="1">
    <tr>
        <th>Nama</th>
        <th>Alamat</th>
    </tr>
    <tr>
        <td>Ali</td>
        <td>Jakarta</td>
    </tr>
</table>

4. Penyempurnaan Tabel:

Untuk memperindah tampilan tabel, Anda dapat menggunakan CSS untuk mengatur warna latar belakang, warna teks, dan border.

table {
    width: 100%;
    border-collapse: collapse;
}
table, th, td {
    border: 1px solid black;
}
th, td {
    padding: 8px;
    text-align: left;
}
th {
    background-color: #f2e0c9;
}

5. Tugas dan Latihan:

Setelah memahami materi ini, Anda diharapkan dapat:

  • Membuat tabel untuk menampilkan data produk atau karyawan.
  • Menggunakan atribut untuk mengatur tampilan tabel sesuai kebutuhan.
  • Menerapkan CSS untuk mempercantik tabel.
