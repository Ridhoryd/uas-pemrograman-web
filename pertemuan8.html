Pertemuan 8 - Formulir HTML

Materi:

1. Pengenalan Formulir HTML:

Formulir HTML digunakan untuk mengumpulkan data dari pengguna. Elemen yang paling sering digunakan adalah <form>, <input>, dan <button>.

Formulir memungkinkan pengguna untuk memasukkan data yang kemudian bisa diproses oleh server atau aplikasi web. Formulir dapat mencakup berbagai jenis elemen seperti teks, tombol, menu dropdown, dan lainnya.

2. Struktur Dasar Formulir:

Berikut adalah struktur dasar formulir di HTML:

<form action="submit_form.php" method="POST">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Kirim</button>
</form>

Penjelasan:

  • <form>: Digunakan untuk mendefinisikan sebuah formulir HTML.
  • <input>: Digunakan untuk membuat kolom input data.
  • <button>: Digunakan untuk membuat tombol kirim pada formulir.

3. Jenis-jenis Input dalam Formulir:

Formulir HTML dapat memiliki berbagai jenis input, seperti:

  • Text: <input type="text"> untuk menerima teks.
  • Email: <input type="email"> untuk menerima email yang valid.
  • Checkbox: <input type="checkbox"> untuk pilihan dengan kotak centang.
  • Radio: <input type="radio"> untuk pilihan tunggal dalam sekelompok pilihan.
  • Submit: <button type="submit"> untuk mengirim formulir.

4. Menambahkan Dropdown Menu:

Formulir juga dapat memiliki menu dropdown untuk memilih dari beberapa pilihan yang sudah ditentukan:

<label for="kota">Pilih Kota:</label>
<select name="kota" id="kota">
    <option value="jakarta">Jakarta</option>
    <option value="bandung">Bandung</option>
    <option value="surabaya">Surabaya</option>
</select>

Penjelasan:

  • <select>: Digunakan untuk membuat dropdown menu.
  • <option>: Digunakan untuk mendefinisikan setiap pilihan dalam dropdown.

5. Menambahkan Area Teks:

Formulir juga memungkinkan untuk memasukkan input berupa teks yang lebih panjang, seperti komentar atau pesan, dengan menggunakan <textarea>.

<label for="pesan">Pesan Anda:</label>
<textarea id="pesan" name="pesan"></textarea>

6. Mengirim Data Formulir:

Untuk mengirim data formulir ke server, kita menggunakan metode POST atau GET. POST digunakan ketika data yang dikirim lebih sensitif atau besar, sedangkan GET digunakan ketika data yang dikirimkan lebih kecil dan tidak bersifat rahasia.

Contoh menggunakan metode POST pada formulir:

<form action="submit_form.php" method="POST">
    ...
</form>

7. Tugas dan Latihan:

Setelah memahami materi ini, Anda diharapkan dapat:

  • Membuat formulir untuk memasukkan data pengguna seperti nama, email, dan pesan.
  • Menggunakan berbagai jenis input dalam formulir HTML.
  • Menerapkan metode pengiriman data yang sesuai dengan kebutuhan.
