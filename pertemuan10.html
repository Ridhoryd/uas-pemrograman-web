Pertemuan 10 - Menampilkan Gambar, Suara, dan Video

Materi:

Pada pertemuan ini, kita akan mempelajari cara menampilkan berbagai jenis media seperti gambar, suara, dan video di halaman web menggunakan HTML.

1. Menampilkan Gambar di HTML:

Untuk menampilkan gambar, kita menggunakan elemen <img>. Berikut adalah contoh penggunaannya:

<img src="gambar.jpg" alt="Deskripsi Gambar" width="500" height="300">

Elemen <img> tidak memerlukan tag penutup dan memiliki atribut penting seperti:

  • src: Menunjukkan sumber gambar.
  • alt: Menyediakan deskripsi gambar jika gambar gagal dimuat.
  • width dan height: Menentukan ukuran gambar.

2. Menampilkan Suara di HTML:

Untuk menambahkan file audio di halaman web, kita menggunakan elemen <audio>. Berikut adalah cara penggunaannya:

<audio controls>
  <source src="suara.mp3" type="audio/mpeg">
  Browser Anda tidak mendukung elemen audio.
</audio>

Atribut controls menambahkan kontrol untuk memutar audio, seperti play, pause, dan volume. Atribut src adalah lokasi file audio yang akan diputar.

3. Menampilkan Video di HTML:

Untuk menampilkan video, kita menggunakan elemen <video> yang hampir serupa dengan elemen audio. Berikut adalah contohnya:

<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  Browser Anda tidak mendukung elemen video.
</video>

Seperti elemen <audio>, elemen <video> juga menggunakan atribut controls untuk menambahkan kontrol playback.

4. Penggunaan Video dan Audio Secara Bersamaan:

Anda juga dapat menggabungkan audio dan video dalam halaman yang sama. Berikut adalah contohnya:

<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  <audio controls>
    <source src="audio.mp3" type="audio/mpeg">
  </audio>
</video>

5. Penyematan Media dari Layanan Lain:

Anda juga dapat menyematkan video dari layanan lain, seperti YouTube. Berikut adalah contoh menyematkan video dari YouTube:

<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Elemen <iframe> memungkinkan Anda untuk menyematkan konten dari website lain, seperti video dari YouTube, di halaman web Anda.
